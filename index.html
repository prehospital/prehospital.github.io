About

Prehospital is an open source site which aims to collaborate the best learning resources for budding paramedics. The ethos of which everyone is free to contribute.

At present the site features some useful printouts to take with you on the road as well as collection of learning resources.

If you are tech-savvy and interested in the intersection between pre-hospital care and open source software you will find the source code at http://github.com/prehospital/prehospital.github.io
